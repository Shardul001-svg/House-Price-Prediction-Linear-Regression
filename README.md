# House Price Prediction using Linear Regression

## Project Overview

This project predicts house prices using the California Housing Dataset and a Linear Regression Machine Learning model.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Dataset

California Housing Dataset from Scikit-Learn.

## Steps Performed

1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing
4. Train-Test Split
5. Linear Regression Model Training
6. Model Evaluation
7. Visualization of Results

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

* task1_ml_linear_regression.ipynb
